Abstract:

Para que possamos receber um pedido pelo url, usamos o nodejs com o modulo http. Com essa informação vamos poder abrir o ficheiro pedido pelo url.

Usando os conhecimentos anteriores de xsl, o conhecimento do <xsl:copy>, para facilitar o processo, o conhecimento de count() e de eixos do XPath, conseguimos construir vários ficheiros com contagem crescente.

O exercício de mostrar o ficheiro específico em nodejs foi muito simples tendo o exemplo da aula anterior.

Deliverables

  • pr.xsl - stylesheet xml->html
  • pr.html - resultado do xslt do pr
  • arq.xsl - stylesheet xml->(vários documentos em arq_sit)
  • arq_sit - diretoria com o resultado de xslt do arq (arq1.xml, arq2.xml, ...)
  • server.js - servidor para dado um i no localhost:7777/i, abre o arq_sit/arqi.xml
