ABSTRACT:

Com DTDs consegue-se uma forma de validar se os XMLs estão num formato esperado. Algo muito útil para que não haja surpresas para um programa que só espera que o documento xml está no formato esperado.

Neste trabalho foi pedido que fizéssemos dois DTDs, tanto para este PR como para um XML dado (arq.xml). O método usado foi inspecionar pelo XPath todos os nomes dos elementos em forma de árvore com a shell de xmllint, depois de ter esta base ouve uma iteração com o xmllint --dtdvalid até ter o DTD que estivesse satisfeito.

Também foi pedido que fossem feitas 3 queries, em XPath que correspondem às 3 linhas em arq.txt, foi feito apenas com os conhecimentos transmitidos na aula de 24-09-2019, e com uma pequena verificação usando xmllint --xpath
