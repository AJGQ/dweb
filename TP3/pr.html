Abstract:

Para simplificar e standardizar a escrita de algum tipo de documento, é preferível usar xml, para que seja possível e sem muitas dores de cabeça, a verificação da sua estrutura e também certas transformações como neste caso de xml->html. Sendo também possível de um tipo de estrutura de xml para outro.

Neste trabalho foi criado um stylesheet para transformações xslt (xsl), para transformar este documento e o documento de arqueossítios, num html, no caso do arqueossítios num website, já com várias páginas.

Deliverables

  • pr.xsl (stylesheet xml->html)
  • pr.html (resultado do xslt do pr)
  • arq.xsl (stylesheet xml->html/website)
  • website (resultado do xslt do arq)
